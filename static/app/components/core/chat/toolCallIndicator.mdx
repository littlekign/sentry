The `ToolCallIndicator` communicates the aggregate status of a message's tool
calls in an agent conversation: a spinner while they run and a semantic icon
once they settle.

## Statuses

  <Grid columns="16px 1fr" flow="row" gap="md" align="center">

    <Text>loading — work in progress</Text>
