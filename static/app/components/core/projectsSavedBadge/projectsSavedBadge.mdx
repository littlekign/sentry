import {Flex} from '@sentry/scraps/layout';
import {ProjectsSavedBadge} from '@sentry/scraps/projectsSavedBadge';
import {Text} from '@sentry/scraps/text';

import * as Storybook from 'sentry/stories';

export const documentation = import('!!type-loader!@sentry/scraps/projectsSavedBadge');

`ProjectsSavedBadge` renders a fixed 16×16 badge of the project platform icons associated
with a saved view — a starred project, saved query, dashboard, or issue view. It's used by
secondary-navigation items and breadcrumb crumbs to give each a recognizable glyph while
keeping every leading slot the same size, so labels stay aligned.

## Usage

```tsx
import {ProjectsSavedBadge} from '@sentry/scraps/projectsSavedBadge';
```

The badge adapts to how many platform slugs you pass in `projectPlatforms`:

- **0 platforms** — a "my projects" glyph, or an "all projects" glyph when `allProjects`
  is set.
- **1 platform** — a single bordered platform icon.
- **2+ platforms** — the first two platforms as stacked icons (extra platforms are not
  shown).

<Storybook.Grid>
  <Flex align="center" gap="sm">
    <ProjectsSavedBadge projectPlatforms={[]} />
    <Text variant="muted">My projects (0)</Text>
  </Flex>
  <Flex align="center" gap="sm">
    <ProjectsSavedBadge projectPlatforms={[]} allProjects />
    <Text variant="muted">All projects (0)</Text>
  </Flex>
  <Flex align="center" gap="sm">
    <ProjectsSavedBadge projectPlatforms={['javascript']} />
    <Text variant="muted">1 platform</Text>
  </Flex>
  <Flex align="center" gap="sm">
    <ProjectsSavedBadge projectPlatforms={['python', 'javascript']} />
    <Text variant="muted">2 platforms</Text>
  </Flex>
  <Flex align="center" gap="sm">
    <ProjectsSavedBadge projectPlatforms={['react', 'node', 'python']} />
    <Text variant="muted">3 platforms (first two shown)</Text>
  </Flex>
</Storybook.Grid>
```tsx
{/* No projects selected */}
<ProjectsSavedBadge projectPlatforms={[]} />
{/* All projects */}
<ProjectsSavedBadge projectPlatforms={[]} allProjects />
{/* A single project */}
<ProjectsSavedBadge projectPlatforms={['javascript']} />
{/* Multiple projects — first two platforms are stacked */}
<ProjectsSavedBadge projectPlatforms={['python', 'javascript']} />
```

Platform slugs are passed straight through to `PlatformIcon` from `platformicons`, so any
supported platform renders:

<Storybook.Grid>
  {['javascript', 'python', 'ruby', 'php', 'java', 'go', 'rust', 'node'].map(platform => (
    <Flex key={platform} align="center" gap="sm">
      <ProjectsSavedBadge projectPlatforms={[platform]} />
      <Text variant="muted">{platform}</Text>
    </Flex>
  ))}
</Storybook.Grid>

## Accessibility

The badge renders `aria-hidden` — it's decorative, and its meaning is carried by the label
it sits beside.

### Developer responsibilities

- Always pair `ProjectsSavedBadge` with a visible text label. Never rely on the badge alone
  to convey which saved view, project, or item a crumb or link points to.
